Install Android

Before add the jar Honey Badger, you need to set your Android environment, you can set this following the next link: http://developer.android.com/sdk/index.html


Download

You can download the Honey Badger from this link: Honey Badger 1.0 - beta and add to your class path the jar. If you had any problem running your project cause of Badger-1.0.jar, make this changes in your project.
